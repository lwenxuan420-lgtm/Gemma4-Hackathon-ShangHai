📂 共享映射文档管理 SYNC-BRIDGE READY

为所有模块统一绑定共享盘 / 网络映射文件（XLSX / XLSM / CSV / JSON / HTML）。
绑定即永久有效，关闭浏览器再打开、嵌入集成看板均自动恢复，直到你主动点「解绑」。
推荐：点下方"一键绑定维修明细"，将同一份 XLSX 映射到全部4个图表模块。

模块 已绑定文件 刷新间隔(秒) 操作
使用说明：
• 绑定即永久：使用 Edge / Chrome，点「绑定文件（永久）」后记录写入 IndexedDB，关闭浏览器再打开自动恢复，无需重新选文件。
• 只有主动点击「解绑」才会清除绑定，否则终身有效。
• 嵌入集成看板（iframe）时，直接在子面板内点绑定，子页面自行弹选文件框，不再依赖父页代理，不会超时。
• 首次绑定后浏览器可能弹「允许读取文件」权限提示，点允许即可；之后刷新无需再次授权。
